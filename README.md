# Desafio_Angular_Ford_Enter

Para quem for executar o sistema, instruções:

API (back‑end):

Navegue até a pasta da API.

Execute npm install express cors (se ainda não tiver).

Execute node api.js – a API ficará em http://localhost:3001.

Front‑end (Angular):

Navegue até a pasta do projeto Angular.

Execute npm install (se necessário, para instalar dependências).

Execute ng serve – o sistema ficará em http://localhost:4200.

Acesse e faça login com admin / 123456.

Imagens:

Coloque as imagens (logo da Ford, veículos) em public/img/.
